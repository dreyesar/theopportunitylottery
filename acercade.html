Este es un juego en el cual debes acertar las tres figuras para ganar. Cada una tendrá una puntuación distinta, siendo el rey la de mayor puntaje con 6 estrellas de premio. El sistema al azar asignará monedas cada vez que refresques la página o presiones sobre Home. Verás que cada vez que ganes o pierdas monedas se veran reflejas bajo el número total.
